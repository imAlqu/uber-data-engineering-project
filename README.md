# Uber Data Engineering Project

## Overview
An end-to-end data engineering project built using Python, Google BigQuery and Looker Studio.

## Technologies Used
- Python (Pandas)
- Google BigQuery
- Looker Studio
- GitHub

## Dataset
Uber NYC Trip Data with 100,000 records

## Project Steps
1. Loaded and cleaned data using Python
2. Created dimension tables and fact table
3. Uploaded data to Google BigQuery
4. Created analytics report using SQL
5. Built dashboard in Looker Studio

## Files
- Uber Data Pipeline.ipynb - Main notebook
- analytics_query.sql - BigQuery SQL query
